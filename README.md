# CapyUNI Codium (Web)

A lightweight, persistent Web IDE inspired by modern code editors. Built with React, TypeScript, and Monaco Editor.

## Features

### Stage 1: Core
-   **Visual Fidelity**: Custom layout matching popular IDEs.
-   **File System**: Create, rename, delete files/folders.
-   **Editor**: Monaco Editor with syntax highlighting.
-   **Persistence**: IndexedDB auto-save.

### Stage 2: Advanced
-   **Search**: Global text search.
-   **Quick Open (Ctrl+P)**: Fuzzy file finder.
-   **Terminal**: Real xterm.js integration.

### Stage 3: Pro Modules (NEW)
-   **Import/Export**: Save your workspace as a `.zip` file and load it back on any device.
-   **Capy Chat (Gemini AI)**:
    -   Integrated AI assistant (sidebar).
    -   Context-aware coding help.
    -   **"Apply Changes"**: Safe mode that lets you review and apply code generated by AI directly to your files.
    -   *Requires Google Gemini API Key*.
-   **Capy Modules**:
    -   **Docs**: Auto-generate markdown documentation structure.
    -   **Community**: One-click boilerplate templates.
    -   **Blog**: Read playbooks and import them as guides into your workspace.

## Tech Stack

-   React 18 + TypeScript + Vite
-   Tailwind CSS + Zustand
-   Monaco Editor + xterm.js
-   JSZip + FileSaver (Import/Export)
-   Google Gemini REST API

## Shortcuts

| Shortcut | Action |
| - | - |
| **Ctrl+B** | Toggle Sidebar |
| **Ctrl+P** | Quick Open File |
| **Ctrl+Shift+P** | Command Palette |
| **Ctrl+`** | Toggle Terminal |

## How to use AI
1.  Click the Robot icon in the Activity Bar.
2.  Click the Key icon to paste your Gemini API Key (stored locally).
3.  Ask "Refactor this file to use React Hooks".
4.  If Capy proposes code, click "Apply Changes" to update your file.

## How to Run
```bash
npm install
npm run dev
```
